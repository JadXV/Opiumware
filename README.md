# Opiumware

Opiumware is a macOS executor UI built with Tauri. It features a full code editor, AI assistant, script hub, autoexec, and RConsole integration.

## Installation

Run the following command in Terminal:

```bash
curl -fsSL https://raw.githubusercontent.com/JadXV/Opiumware/main/install.sh | bash
```

## Manual Installation

1. Download the latest release from [GitHub Releases](https://github.com/JadXV/Opiumware/releases)
2. Extract the ZIP file
3. Move Opiumware.app to your Applications folder
4. Run `xattr -rd com.apple.quarantine /Applications/Opiumware.app`
5. Launch Opiumware from your Applications folder

## Features

- Code editor with syntax highlighting, Luau LSP diagnostics, and intellisense
- OpiumwareAI, a built-in AI assistant for scripting help
- Script Hub with ScriptBlox integration
- Local script management with folder support
- Autoexec support
- RConsole with live Roblox log monitoring
- Multiple theme options
- Multi-instance execution with port selection
